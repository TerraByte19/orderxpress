Wie kommen Gäste auf ihre Seite?

Jeder Tisch hat einen QR-Code. In der Inhaber-Ansicht bei einem Tisch auf „QR“ klicken (zum Ausdrucken) oder auf „Gast-Ansicht“, um die Seite direkt im Browser auszuprobieren.
